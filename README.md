# 世界杯量化决策台

轻量化单页应用，覆盖 2026 世界杯 104 场赛程。每场对阵带本地国旗，并提供 Poisson 比分分布、三个建议比分、胜平负概率、让球重算和阵容可用度修正。

## 启动

```powershell
npm start
```

浏览器打开 `http://127.0.0.1:4173`。

## 竞彩计算器

- 支持胜平负与比分两种玩法独立切换。
- 比分模式包含官方 31 项比分选项，并标注模型概率最高的三个建议比分。
- 比分建议会参考最近两个比赛日的已完赛比分，对同队后续场次做小幅进攻/防守状态校准。
- 支持胜、平、负以及比分单选与复式选择；胜平负最多 8 场，比分最多 4 场。
- 支持自由过关和完整的 `M 串 N` 固定组合。
- 注数按每注 2 元计算，投注倍数限制为 1–50 倍。
- 单注奖金按固定奖金连乘后执行“四舍六入、五成双”保留两位。
- 单注封顶：单关 10 万元，2–3 关 20 万元，4–5 关 50 万元，6 关及以上 100 万元。
- 奖金区间枚举所有已选结果的命中情景，实际奖金以出票时固定奖金及兑奖规则为准。

## 数据与边界

- 赛程：公开 2026 世界杯赛程 JSON，本地保存稳定快照。
- 赛果：OpenFootball `worldcup.json`，启动时尝试在线刷新。
- 模型：参考 2026 年 6 月 11 日 FIFA 男足排名基线，结合复合实力分、场地主队修正、阵容可用度、近两日完赛比分与 Poisson 进球分布。
- 淘汰赛对阵产生前只显示赛程占位，不伪造具体球队或比分。
- 所谓资本操作、假球或演戏必须有正式调查证据，未经证实的信息不进入模型。
- 本工具用于概率教育与情景分析，不提供“稳赚”“必中”或确定收益。
